About This Page

This sample page demonstrates how to use semantic HTML5 tags, improve accessibility, and apply basic SEO principles without using any CSS or JavaScript.

Accessibility Best Practices

Using semantic tags like <header>, <nav>, and <main> helps screen readers understand your page structure. Provide alt text for images and use aria attributes when needed.

SEO Tips for Beginners

  • Use a single <h1> as your page title.
  • Organize headings in a clear, logical order (h1 → h2 → h3).
  • Write descriptive, human-readable content.
  • Include relevant keywords in headings and paragraphs naturally.
  • Use the <meta name="description"> tag to summarize your page.
